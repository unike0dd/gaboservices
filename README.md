# Gabriel - Professional Business Services

This repository contains the website for Gabriel's professional business services.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Create a production build:
   `npm run build`
4. Preview the production build:
   `npm run preview`
