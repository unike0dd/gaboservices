# Gabriel Professional Business Services (HTML/CSS/JS)

This repository now uses a plain web stack:
- `index.html`
- `styles.css`
- `main.js`

## Run locally

Prerequisite: Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```
3. Build production assets:
   ```bash
   npm run build
   ```
4. Preview production build:
   ```bash
   npm run preview
   ```
